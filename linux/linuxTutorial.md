# Linux stuff tutorial

## Config nano for a better version

```bash
sset linenumbers

## Every row has number
set softwrap
## Arrange the tab size
set tabsize 4

set autoindent
## Using the mouse to move the cursor
set mouse

set smooth
## Remember the cursor position in each file for the next editing session.
set positionlig

## Turn off the helpful shortcut
set nohelp

## It displays a more compact status bar at the bottom of the screen
set minibar

## (tabs and spaces) instead of always at the edge of the screen
set atblanks

## Automatically hard-wrap the current line when it becomes overlong.
set breaklonglines

set constantshow

## Paint the interface elements of nano.
set promptcolor lightwhite,grey
set statuscolor bold, red, red
set errorcolor bold,white,red
set spotlightcolor black,lightyellow
set selectdcolor lightwhite,magenta
set stripecolor ,yellow
set scrollercolor cyan
set numbercolor cyan
set keycolor cyan
set funnctioncolor green

```
